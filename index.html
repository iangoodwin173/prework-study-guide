HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The user will not see anything within the head element
  • HTML functions as the building blocks of a webpage

CSS

  • CSS is used to give a webpage more style and personality
  • CSS can be scripted into the main HTML form or linked as a separate style sheet
  • margins and padding specify how much space we want around the outside or inside of an element, respectively

Git

  • Git is the main flow that developers use to keep track of their code versions
  • Git (and GitHub) can be used to locally edit and merge remote code
  • Working on Git branches allows multiple people to make changes to the same code without jeopardizing the functionality of the code
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript gives a webpage its interactive properties
